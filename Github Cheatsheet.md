## Github
### Github란?
- 분산 버전 관리 툴인 깃(Git)을 사용하는 프로젝트를 지원하는 웹호스팅 서비스이다.
- 영리적인 서비스와 오픈소스를 위한 무상 서비스를 모두 제공한다.
- 깃(Git)이 텍스트 명령어 입력 방식인데 반해, 깃허브는 화려한 그래픽 유저 인터페이스(GUI)를 제공한다.
